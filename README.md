# Chatbot AI Layanan Informasi Mahasiswa

## Deskripsi
Chatbot AI Layanan Informasi Mahasiswa merupakan aplikasi chatbot berbasis teks yang dirancang untuk membantu mahasiswa dalam memperoleh informasi akademik secara cepat dan efisien. Chatbot ini memanfaatkan konsep Artificial Intelligence (AI) sederhana dengan pendekatan rule-based untuk memproses input pengguna dan memberikan respon yang sesuai. Projek ini dikembangkan sebagai implementasi peran AI di bidang komunikasi serta sebagai pemenuhan tugas Team-Based Project UAS.

---

## Tujuan Projek
Tujuan dari pembuatan projek ini adalah:
- Menerapkan konsep Artificial Intelligence (AI) dalam bidang komunikasi
- Mengembangkan chatbot sederhana sebagai media komunikasi digital
- Membantu mahasiswa memperoleh informasi akademik secara otomatis
- Melatih kerja sama tim dalam pengembangan proyek berbasis GitHub
- Memenuhi tugas Team-Based Project UAS

---

## Peran AI dalam Bidang Komunikasi
Dalam projek ini, AI berperan sebagai:
- Media komunikasi interaktif antara sistem dan pengguna
- Pengolah input teks dari mahasiswa
- Pemberi respon otomatis berdasarkan data pertanyaan dan jawaban

Chatbot ini menunjukkan bagaimana AI dapat menggantikan komunikasi manual menjadi komunikasi digital yang lebih cepat dan efisien.

---

## Cara Menjalankan Chatbot AI Layanan Informasi Mahasiswa
Ikuti langkah-langkah berikut untuk menjalankan Chatbot AI Layanan Informasi Mahasiswa:
1. Pastikan Python sudah terinstall di komputer. Untuk mengecek, jalankan perintah: '''bash python --version 
2. Unduh atau clone repository projek ke komputer
3. Buka terminal atau commond prompt pada folder projek
4. Jalankan program chatbot dengan perintah: python src/chatbot.py
5. Setelah program berjalan, chatbot akan menampilkan pesan pembuka. Pengguna dapat mengetik pertanyaan yang tersedia, seperti jadwal kuliah, cara bayar ukt, pengisian krs, beasiswa, dan nilai akhir
6. Untuk mengakhiri percakapan, ketik exit
